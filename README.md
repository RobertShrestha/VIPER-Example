# VIPER Example

## Getting Started
### Installation or Setup
1. Just download the repo or clone the repo
2. Open Terminal and go inside the folder,e.g cd [path to inside the repo folder]
3. Type pod install
4. Go to the folder,Just open the .xcworkspace 
5. Run the project
